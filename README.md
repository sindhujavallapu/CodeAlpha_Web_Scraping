# CodeAlpha_Web_Scraping# CodeAlpha - Web Scraping Project

## Project Title
Web Scraping of Books Data Using Python and BeautifulSoup

## Objective
The objective of this project is to extract book information from a website and create a structured dataset for analysis.

## Website Used
https://books.toscrape.com/

## Tools and Libraries Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Google Colab

## Project Workflow
1. Connected to the website using Python requests.
2. Parsed the HTML using BeautifulSoup.
3. Extracted book titles and prices.
4. Stored the extracted data in a Pandas DataFrame.
5. Exported the data to a CSV file.

## Output
The project generates a dataset containing:
- Book Title
- Price

## Files Included
- CodeAlpha_Web_Scraping.ipynb
- Books_Data.csv
- README.md

## Learning Outcomes
- Understanding web scraping concepts
- Working with HTML structure

- Creating datasets using Pandas
